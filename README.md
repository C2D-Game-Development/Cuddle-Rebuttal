# Controls

### Jeffrey - the murderous dog

```
arrowLeft = moveLeft
arrowRight = moveRight
arrowUp = jump

arrowDown = shield
j = punchee
l = blood blast!
```

### Ted - the homicidal cat

```
a = moveLeft
d = moveRight
w = jump

s = shield
2 = punchee
3 = blood blast!
```
